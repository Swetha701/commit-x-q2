# Commit-X Question 2

## Problem Statement
Given an array of `N` integers, find the elements that appear more than `N/3` times in the array. If no such element exists, return -1.

## Example Test Cases

### Test Case 1
#### Input
```
7
1 2 3 1 1 2 2
```
#### Output
```
1 2
```

### Test Case 2
#### Input
```
5
1 2 3 4 5
```
#### Output
```
-1
```
